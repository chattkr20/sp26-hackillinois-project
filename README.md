## Running the Project Locally

### Running Backend Locally

```
cd backend
```

#### Windows

```
./start.ps1
```

#### Mac / Linux

```
chmod +x start.sh
./start.sh
```

### Running Frontend Locally

```
cd frontend
npm i
npm run dev
```

It is recommended you use **Chrome** to run the localhost
